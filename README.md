# Conversor de Monedas en C
### Descripción
Una aplicación de consola desarrollada en lenguaje C que permite realizar conversiones de divisas entre Quetzales (GTQ), Dólares (USD) y Euros (EUR) utilizando tasas de cambio definidas.

### Prerrequisitos
* Sistema operativo Linux (Ubuntu Server recomendado).
* Compilador GCC (`sudo apt install gcc`).

### Pasos para su ejecución
1. Abrir la terminal en la carpeta del proyecto.
2. Compilar el código: `gcc conversor.c -o conversor`
3. Ejecutar el programa: `./conversor`
